* **2.1.1. Install & configure VirtualBox**
    - ![Cheat Sheet VirtualBox](../../src/docs/cheat-sheets/cheat-sheet-virtualbox.md)

    - Install VB
    
* **2.1.2. Install Ubuntu VM**
    - ![Cheat Sheet Ubuntu VM](../../src/docs/cheat-sheets/cheat-sheet-virtualbox.md)
    - Download LTS Version of Ubuntu
    - Create Ubuntu VM in VB:
        - Start VB & create new VM [TECSA_DEV] of Type "Linux" & Version "Ubuntu (64-bit)"
        - Memory Allocation: For our dev purposes, a minimum of 4 GB is required. A safe practice is to allocate up to 25% of your machine's total RAM, and if you have 32 GB or more, allocating 8 GB is recommended.
        - CPU Allocation: For development, we require a minimum of 2 CPU cores. A safe practice is to allocate up to 25% of your machine's total cores, and if your host has more than 12 cores, allocating 4 is a solid choice.
        - Choose Ubuntu ISO-File
        - "Create a Virtual Hard Disk" of Type "VDI (VirtualBox Disk Image)" & "Dynamically allocated" & Size "min 25GB" (recommanded 50GB)
        - Start Installation & follow instructions.

* **2.1.3. Otimize ubuntu VM configuration**
